SCOPE: This tool is a Datasheet Completeness & Administrative Compliance Checklist for API 610 Ed.12 (field presence, clause tagging, and a limited set of clause-sourced numeric checks: NPSH margin, MAWP vs §6.3.6 discharge+10%ΔP component, Table 10 bearing-type energy density, Table 18 lateral-analysis logic, hydrotest = 1.5×MAWP). It is NOT a substitute for full mechanical/rotordynamic verification — nozzle loads (Table 5/Annex F), vibration limits (Table 8/9), running clearances (Table 6), bearing L10h life (§6.10.1.11), casing pressure-boundary design, and rotor lateral analysis results (Annex I) still require engineering review by a qualified rotating-equipment engineer. A few clause-number tags are marked "(unverified)" because the source PDF text lost numeric sub-clause prefixes on many bulleted "shall" statements — cross-check those against an officially numbered copy before final sign-off.
