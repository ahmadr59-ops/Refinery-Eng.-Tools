A.Batch Calculation — Line List Import

Calculate wall thickness for every line in a Line List in one pass. Each line's material can come either directly from a Material Spec/Grade column in the Line List itself, or — if the Line List only carries a Piping Class code — from a PMS file that maps Class → Material/Corrosion Allowance/Joint Type.

Material Database

Materials with a source citation were extracted directly from the ASME B31.3-2024 Table A-1 pages you supplied (Basic Allowable Stresses in Tension — SI Units), with the source Line No. and page cited in the Reference column. The S-Table column holds the full temperature-dependent stress curve as it appears in the standard; the calculator linearly interpolates between tabulated points for design temperatures that fall between two entries — only the interpolation method is added, not the tabulated values themselves. Rows with an empty S-Table are placeholders — add data or supply the relevant standard page to populate them.

Scope & limitations: This module implements the thin-wall straight-pipe formula of ASME B31.3 §304.1.2(a), Eq. 3a (t = PD / 2(SEW+PY)). It does not implement the thick-wall equation §304.1.2(b) — the tool will flag and stop if your inputs fall into that regime. It does not cover bends, miters, branch reinforcement, or external pressure (collapse). The Schedule-matching table covers NPS 1/2"–24" per ASME B36.10M; larger or non-standard sizes show OD for reference but require manual schedule selection. This is a personal calculation aid; it is not a substitute for review and sign-off by a qualified engineer against the current, licensed edition of ASME B31.3, and applicable project specifications.
