Data integrity notice. This tool does not embed ASME B31.3 Table A-1 (allowable stress values). You must enter allowable stress (S) values yourself, sourced directly from your licensed ASME B31.3-2024 copy, in the Material Database below. Joint factor (E), weld strength reduction factor (W), and the Y-coefficient are pre-filled with commonly published reference values as a starting point only — verify every one against Tables A-1B, 302.3.5, and 304.1.1 in your copy before relying on any result for a real deliverable.

Scope & limitations: This module implements the thin-wall straight-pipe formula of ASME B31.3 §304.1.2(a), Eq. 3a (t = PD / 2(SEW+PY)). It does not implement the thick-wall equation §304.1.2(b) — the tool will flag and stop if your inputs fall into that regime. It does not cover bends, miters, branch reinforcement, or external pressure (collapse). This is a personal calculation aid; it is not a substitute for review and sign-off by a qualified engineer against the current, licensed edition of ASME B31.3, and applicable project specifications.
